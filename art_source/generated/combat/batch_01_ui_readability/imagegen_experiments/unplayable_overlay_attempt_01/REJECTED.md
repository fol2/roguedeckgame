# Rejected Experiment - Unplayable Overlay Attempt 01

This generated source is preserved for workflow traceability only.

Decision: do not use a dedicated image overlay for unplayable cards in Batch 01. The engine should render the unplayable state by greying, dimming, or desaturating the whole card.

Reason: an image-generated overlay would add another alignment surface without improving readability. It also risks fighting per-card frame silhouettes. Keep this directory as an abandoned experiment; do not promote it to `public/assets`.
